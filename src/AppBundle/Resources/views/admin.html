XenobladeX.org - Pflege

Generell zur Bedienung:

  • Änderungen können immer über den Button mit Häkchen bestätigt werden
  • Änderungen in einzeiligen Textfeldern können auch mit Druck auf Enter bestätigt
  • Änderungen in mehrzeiligen Textfeldern können auch mit STRG + Enter bestätigt
  • Bestätigt man eine Änderung wird die gesamte Zeile gespeichert
  • Um keine Änderung vorzunehmen kann man einfach mit Druck auf ESC das Feld wieder verlassen
  • Ausgenommen der Dateiverwaltung kann immer die zuletzt gelöschte Zeile wieder gergestellt werden
  • nach Löschen taucht oberhalb der Pflege eine Meldung auf, um die Zeile wiederherzustellen
  • Eine gelöschte und anschließend wiederhergestellte Zeile kann erst nach einem Reload erneut gelöscht werden.
