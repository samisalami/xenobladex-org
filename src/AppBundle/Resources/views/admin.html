XenobladeX.org - Pflege

Generell zur Bedienung:

  • Änderungen können immer über den Button mit Häkchen bestätigt werden
  • Änderungen in einzeiligen Textfeldern können auch mit Druck auf Enter bestätigt werden, in mehrzeiligen Textfeldern mit STRG + Enter
  • Bestätigt man eine Änderung wird die gesamte Zeile gespeichert
  • Neue Einträge werden grundsätzlich erst bei Druck auf den Button "Hinzufügen" gespeichert
  • Nicht bestätigte Einträge (Editiermodus noch geöffnet) werden nicht gespeichert
  • Um keine Änderung vorzunehmen kann man auch mit Druck auf ESC das Feld wieder verlassen
  • Ausgenommen der Dateiverwaltung kann immer die zuletzt gelöschte Zeile wieder gergestellt werden
  • nach Löschen taucht oberhalb der Pflege eine Meldung auf, um die Zeile wiederherzustellen
