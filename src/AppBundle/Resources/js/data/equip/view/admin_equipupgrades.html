Stufen pflegen:
  • Da es sich um stark verschachtelte Daten handelt gibt es mehrere "Speicher"-Buttons:
    Einen zum Speichern der übergeordneten Erweiterung, einen zum Speichern einer Stufe (hinter Verkaufswert) und einer zum Speichern der verknüpften Materialien
  • Stufen können nur an bereits angelegten Erweiterungen hinzugefügt werden
  • Öffnet ihr eine bestehende Erweiterung, welche noch keine Stufen hat, könnt ihr diese über einen Button erzeugen (Stufen anlegen)
  • Sollte beim Anlegen irgendetwas schief gegangen sein, so dass ihr nicht die 5 Stufen habt, aber auch nicht mehr den Button "Stufen anlegen", so gibt es den Button "Stufen löschen" um sie neu generieren zu können (es werden nur Stufen gelöscht, an denen kein Material hängt, dieses ggf. händisch löschen)
